# Train-Ticket-Management-System
A C++ console application for managing train ticket reservations using linked lists.

## Features

- Book tickets
- Search tickets
- Cancel tickets
- Display all booked tickets
- Count booked tickets
- Prevent duplicate seat booking

## Technologies

- C++
- Object-Oriented Programming
- Linked Lists
- Dynamic Memory Allocation

## How to Run

Compile:

```bash
g++ TrainTicketManagement.cpp -o train
```

Run:

```bash
./train
```
